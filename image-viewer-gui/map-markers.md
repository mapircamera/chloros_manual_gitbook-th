# เครื่องหมายแผนที่

แท็บแผนที่จะแสดงภาพของคุณบนแผนที่ 2 มิติเชิงโต้ตอบตามพิกัด GPS ข้อมูลนี้ให้ภาพรวมทางภูมิศาสตร์ของเซสชันการจับภาพของคุณและช่วยให้คุณเห็นภาพความครอบคลุมเชิงพื้นที่ นอกจากนี้ยังมีประโยชน์เมื่อนำเข้ารูปภาพของคุณเป็นครั้งแรก เพื่อลบรูปภาพใดๆ ที่คุณไม่จำเป็นต้องประมวลผลออกอย่างรวดเร็ว

<figure><img src="../.gitbook/assets/chloros_map_markers.gif" alt=""><figcaption></figcaption></figure>

## การเข้าถึงแท็บแผนที่

1. เปิดหรือสร้างโปรเจ็กต์ใน Chloros
2. นำเข้ารูปภาพที่มีข้อมูลเมตา GPS
3. คลิกแท็บ **แผนที่** <img src="../.gitbook/assets/image (3).png" alt="" data-size="line"> ในแถบด้านข้างซ้าย
4. แผนที่จะแสดงเครื่องหมายที่ตำแหน่ง GPS ของแต่ละภาพ

{% hint style="info" %}
**ต้องใช้ GPS**: เฉพาะภาพที่มีพิกัด GPS ฝังอยู่ในข้อมูลเมตา EXIF เท่านั้นที่จะปรากฏบนแผนที่ ตรวจสอบให้แน่ใจว่ากล้องของคุณเปิดใช้งาน GPS ระหว่างการจับภาพ
{% endhint %}

## เครื่องหมายรูปภาพ

แต่ละภาพที่มีข้อมูล GPS จะแสดงด้วยเครื่องหมายบนแผนที่:

### การแสดงเครื่องหมาย

* เครื่องหมายระบุพิกัด GPS ที่แน่นอนที่แต่ละภาพถูกถ่าย
* เครื่องหมายที่เป็นกลุ่มอาจรวมกลุ่มเข้าด้วยกันเมื่อซูมออก
* ขยายเพื่อดูตำแหน่งภาพแต่ละภาพ

{% hint style="success" %}
ซูมพิเศษ: เมื่อคุณไปถึงระดับการซูมสูงสุดจากผู้ให้บริการส่วนย่อยแผนที่ จากนั้นส่วนย่อยจะถูกขยายใหญ่ขึ้นเมื่อซูมเพิ่มเติม ทำให้คุณเห็นเครื่องหมายที่อยู่ใกล้กัน
{% endhint %}

### ดูตัวอย่างแบบโฮเวอร์

* **เลื่อนเมาส์ของคุณ** เหนือเครื่องหมายใดๆ เพื่อดูตัวอย่างภาพขนาดย่อของภาพนั้น
* ช่วยให้สามารถระบุตัวตนด้วยภาพได้อย่างรวดเร็วโดยไม่ต้องออกจากมุมมองแผนที่
* มีประโยชน์สำหรับการค้นหาภาพเฉพาะภายในเซสชันการจับภาพขนาดใหญ่

***

## ผู้ให้บริการแผนที่ไทล์

{% hint style="success" %}
**การเลือกอัตโนมัติ**: Chloros จะเลือกบริการไทล์ที่ให้ระดับการซูมที่ดีที่สุดสำหรับตำแหน่งแผนที่ปัจจุบันของคุณโดยอัตโนมัติ คุณสามารถสลับระหว่างผู้ให้บริการด้วยตนเองได้หากต้องการ
{% endhint %}

แท็บแผนที่สนับสนุนผู้ให้บริการไทล์สองรายสำหรับภาพแผนที่พื้นหลัง:

### กูเกิลแผนที่

* ภาพถ่ายดาวเทียมและแผนที่มาตรฐานจาก Google
* ดีที่สุดสำหรับความคุ้มครองทั่วไปทั่วโลก

### อีเอสอาร์ไอ

* ภาพถ่ายดาวเทียมและทางอากาศจาก ESRI ArcGIS
* มักให้ภาพที่มีความละเอียดสูงกว่าในบางภูมิภาค

## การนำทางแผนที่

### การควบคุมการซูม

* **ซูมเข้า/ออก**: ใช้ล้อเลื่อนของเมาส์หรือปุ่มซูม
* **เต็มหน้าจอ**: แสดงแผนที่แบบเต็มหน้าจอ

### การควบคุมการแพน

* **แพน**: คลิกและลากเพื่อเลื่อนไปรอบๆ แผนที่

***

## กรณีการใช้งาน

### การแสดงเส้นทางการบิน

* ดูพื้นที่ครอบคลุมของเซสชันการจับโดรน
* ระบุช่องว่างในการครอบคลุมของภาพ
* ตรวจสอบการดำเนินการเส้นทางการบิน

### ทบทวนการสำรวจภาคพื้นดิน

* ดูการกระจายเชิงพื้นที่ของการจับภาพตามภาคพื้นดิน
* ค้นหาภาพเป้าหมายการสอบเทียบที่สัมพันธ์กับพื้นที่สำรวจ
* วางแผนสถานที่จับภาพเพิ่มเติม

### การควบคุมคุณภาพ

* ระบุภาพที่ถ่ายในสถานที่ที่ไม่คาดคิดได้อย่างรวดเร็ว
* ตรวจสอบความแม่นยำของ GPS ทั่วทั้งชุดข้อมูล
* ตำแหน่งภาพอ้างอิงโยงพร้อมบันทึกภาคสนาม

***

## การแก้ไขปัญหา

### ไม่มีเครื่องหมายปรากฏ

**สาเหตุที่เป็นไปได้:**

* รูปภาพไม่มีข้อมูลเมตา GPS
* GPS ถูกปิดใช้งานบนกล้องระหว่างการจับภาพ
* ข้อมูล EXIF ​​ถูกปล้นโดยซอฟต์แวร์ภายนอก

**วิธีแก้ปัญหา**: ตรวจสอบว่าเปิดใช้งาน GPS ในกล้องของคุณและนำเข้าไฟล์ต้นฉบับอีกครั้ง

### เครื่องหมายอยู่ในตำแหน่งที่ไม่ถูกต้อง

**สาเหตุที่เป็นไปได้:**

* กล้อง GPS มีการแก้ไขดาวเทียมที่ไม่ดี
* GPS ดริฟท์ระหว่างการจับภาพ

**วิธีแก้ไข**: โดยทั่วไปนี่เป็นปัญหาเกี่ยวกับเวลาในการจับภาพ พิจารณาใช้ PPK/RTK GPS สำหรับการใช้งานที่มีความแม่นยำ
