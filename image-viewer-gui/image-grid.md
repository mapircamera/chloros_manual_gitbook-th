# ตารางรูปภาพ

หลังจากนำเข้ารูปภาพเข้าสู่โปรเจ็กต์แล้ว คุณจะเห็นรูปภาพที่จัดเรียงเป็นตารางในพื้นที่หลัก

## ขนาดภาพขนาดย่อ

คุณสามารถใช้แถบเลื่อนซูมที่มุมขวาบนเพื่อปรับขนาดภาพขนาดย่อของรูปภาพได้ คุณยังสามารถกด CTRL + ล้อเมาส์เพื่อปรับขนาดได้

## ส่งออกปุ่มมุมมอง

ขณะที่รูปภาพถูกประมวลผล ปุ่มต่างๆ จะปรากฏที่ด้านซ้ายบนของพื้นที่ตาราง ปุ่มเหล่านี้จะสลับภาพขนาดย่อของตารางระหว่างรูปภาพประเภทต่างๆ:

* **JPG** - ภาพขนาดย่อ JPG ของกล้องต้นฉบับ
* **เป้าหมาย** - ภาพที่มีเป้าหมายการปรับเทียบที่ตรวจพบ
* **การสะท้อนแสง** - ภาพการสะท้อนแสงที่ปรับเทียบแล้ว
* **RWA (ดัชนี/LUT)** - ภาพดัชนีหลายสเปกตรัมที่คำนวณพร้อมการไล่ระดับสี LUT

การคลิกที่ภาพขนาดย่อของตารางรูปภาพจะเปิดแบบเต็มหน้าจอ [Image Viewer](opening-an-image-full-screen.md) หากตั้งค่าโหมดการดูบนตารางเป็นการส่งออกเฉพาะ เช่น "เป้าหมาย" เมื่อคลิกภาพขนาดย่อของตาราง ระบบจะเปิดภาพเป้าหมายที่ส่งออกแบบเต็มหน้าจอ
