# คำถามที่พบบ่อย

<รายละเอียด>

<summary>ฉันสามารถประมวลผลภาพจากกล้องที่ไม่ใช่ยี่ห้อ MAPIR ด้วย Chloros ได้หรือไม่</summary>

ไม่ Chloros รองรับการประมวลผลภาพของกล้อง MAPIR เท่านั้น โปรดดูรายการ [รุ่นกล้องที่รองรับ](supported-cameras.md) สำหรับข้อมูลเพิ่มเติม เรามีการประมวลผลของกล้องอื่นๆ บน MAPIR Cloud ดูรายการทั้งหมด [ที่นี่](https://mapir.gitbook.io/mapir-cloud/supported-cameras)

</รายละเอียด>

<รายละเอียด>

<summary>ฉันสามารถปรับเทียบภาพสำหรับการสะท้อนแสงโดยไม่มีเป้าหมายการปรับเทียบได้หรือไม่</summary>

ไม่ หากไม่มีภาพของเป้าหมายการปรับเทียบที่ถ่ายรอบๆ เมื่อถ่ายภาพที่ไม่ใช่เป้าหมาย คุณจะไม่สามารถเชื่อมโยงค่าพิกเซลของภาพกับเปอร์เซ็นต์การสะท้อนแสงที่ทราบได้ หากคุณไม่ได้รวมบันทึกจากเซ็นเซอร์วัดแสง MAPIR ไปด้วย สเปกตรัมแสงโดยรอบจะไม่ถูกวัด และผลลัพธ์การสะท้อนแสงจะไม่แม่นยำ

</รายละเอียด>

<รายละเอียด>

<summary>ฉันสามารถแก้ไขภาพก่อนประมวลผลใน Chloros ได้หรือไม่</summary>

ไม่ Chloros ถือว่าข้อมูลอินพุตไม่ได้รับการแก้ไข ห้ามเปลี่ยนชื่อไฟล์

</รายละเอียด>

<รายละเอียด>

<summary>ฉันสามารถตั้งค่ากล้อง MAPIR Survey3 ให้เป็นค่าแสงอัตโนมัติและประมวลผลภาพใน Chloros ได้หรือไม่</summary>

ไม่ ชุดข้อมูลภาพ Survey3 จะต้องมีค่าแสงคงที่/ล็อค ดังนั้นจึงไม่มีความเร็วชัตเตอร์อัตโนมัติหรือ ISO อัตโนมัติ รูปภาพทั้งหมดของกล้องรุ่นเดียวกันจะต้องมีความเร็วชัตเตอร์และ ISO (ค่าแสง) เท่ากัน

</รายละเอียด>

<รายละเอียด>

<summary>Chloros สามารถประมวลผลหรือวิเคราะห์ภาพออร์โธโมซาอิกได้หรือไม่</summary>

ไม่ รองรับเฉพาะภาพของกล้อง MAPIR แต่ละภาพเท่านั้น ไม่สามารถต่อภาพเช่นแผนที่ออร์โธโมซาอิก

</รายละเอียด>

<รายละเอียด>

<summary>ฉันจะเพิ่มความเร็วขั้นตอนการตรวจจับเป้าหมายของ Chloros ได้อย่างไร</summary>

ในตารางไฟล์เบราว์เซอร์ การเลือกรูปภาพเป้าหมายล่วงหน้าในคอลัมน์ทางขวาจะเป็นการบอกให้ Chloros มองเฉพาะรูปภาพเหล่านั้นเพื่อหาเป้าหมายการปรับเทียบ ซึ่งจะช่วยเร่งการประมวลผลได้อย่างมาก

</รายละเอียด>

<รายละเอียด>

<summary>หากฉันจะอัปโหลดภาพไปยัง <a href="https://www.mapir.camera/collections/software/products/mapir-cloud-subscription">MAPIR Cloud</a> ฉันควรประมวลผลใน Chloros ก่อนที่จะอัปโหลดหรือไม่</summary>

หากคุณวางแผนที่จะอัปโหลดไปยังแพลตฟอร์มการประมวลผลออนไลน์ของเรา [MAPIR Cloud](https://www.mapir.camera/collections/software/products/mapir-cloud-subscription) อย่าแก้ไขภาพก่อนที่จะอัปโหลด คลาวด์จะดำเนินการประมวลผลแบบเดียวกันทั้งหมดและอื่นๆ อีกมากมาย

</รายละเอียด>

<รายละเอียด>

<สรุป>MAPIR จะรองรับฟีเจอร์ X หรือไม่? ฉันหวังว่า MAPIR จะเสนอ X จริงๆ</summary>

เราสนใจรับข้อเสนอแนะเกี่ยวกับผลิตภัณฑ์ของเราอยู่เสมอ หากคุณพบปัญหาเกี่ยวกับผลิตภัณฑ์ของเรา หรือมีข้อเสนอแนะว่าเราจะสามารถปรับปรุงผลิตภัณฑ์ของเราได้อย่างไร โปรด [ติดต่อเรา](https://www.mapir.camera/community/contact) เพื่อแบ่งปันความคิดของคุณ การวิจัยและพัฒนาส่วนใหญ่ของเราได้รับการชี้นำโดยการรับฟังความต้องการที่ใหญ่ที่สุดของลูกค้าของเรา

</รายละเอียด>
