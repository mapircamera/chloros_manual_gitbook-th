# รูปแบบภาพที่ส่งออก

### TIFF (16 บิต)

เอาต์พุตรูปแบบ 16 บิต (ตัวเลขดิจิตอลพิกเซล 0 - 65535) TIFF (.tif)

### TIFF (32 บิต เปอร์เซ็นต์ %)

เอาต์พุตรูปแบบ 32 บิต (pixel float 0.0 - 1.0) TIFF (.tif) ค่าพิกเซลแสดงถึงเปอร์เซ็นต์การสะท้อนแสง โดยที่ 0.0 = 0% และ 1.0 = 100%

### **JPG (8 บิต)**

เอาต์พุตรูปแบบ JPG (.jpg) 8 บิต (ตัวเลขดิจิตอลพิกเซล 0 -255)

### **PNG (8 บิต)**

เอาต์พุตรูปแบบ 8 บิต (ตัวเลขดิจิตอลพิกเซล 0 - 255) PNG (.png) การบีบอัดแบบไม่สูญเสียคุณภาพเหมาะสำหรับการดูเว็บและการแสดงภาพ
